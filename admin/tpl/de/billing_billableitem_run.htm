Rechnungslauf

Der Rechnungslauf wird in regelmäßigen Abständen automatisch ausgeführt. Diese Zeitpunkte können Sie in den Einstellungen festlegen.

An dieser Stelle können Sie den Rechnungslauf manuell ausführen. Dabei werden alle abrechnungsfähigen Posten in Rechnungen gewandelt und den jeweiligen Empfängern zugestellt.
