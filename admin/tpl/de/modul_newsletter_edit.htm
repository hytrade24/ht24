Über dieses Tool versendete Newsletter werden automatisch im HTML- Modus verschickt. Daher können Sie auch hier den grafischen Editor zum Verfassen Ihrer Newsletter nutzen.

Entscheiden Sie sich bei "Versandliste" für die Option "alle Newsletter Empfänger", wird der Newsletter beim Versenden an alle Benutzer versendet, welche sich in Ihren Newsletter eingetragen haben. Die Option "Alle Benutzer im System" nimmt auf die Wünsche der Benutzer kein Rücksicht. Jeder bekannte Benutzer wird diesen Newsletter bekommen. Sie sollten diese Option nur nutzen, wenn es sich bei Ihrem Newsletter um eine wichtige Nachricht handelt, da Sie sonst Gefahr laufen Ihre Benutzer zu verärgern!

Bitte beachten Sie, dass der Newsletter noch nicht versendet wird, wenn Sie ihn speichern. Details hierzu finden sie unter Sendeberichte
