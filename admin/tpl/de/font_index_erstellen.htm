Font-Index erstellen

Wie mit dem Bilder-Index, verhält es sich hier mit dem Font-Index. Damit immer eine aktuelle Auswahl der Schriftarten vorliegt, müssen Sie diese Seite immer aufrufen, nachdem Sie Schriftarten (.ttf) aus dem Ordner fonts/ gelöscht, oder hinzugefügt haben. Wenn Sie unten das Wort "done" und außer den Statusmeldungen keine Fehlernachrichten erscheinen, ist die Schriftauswahl aktualisiert.

{if !arfonts} {anfang} {dirname} read directories ...
{error}

done

{else} {arfonts} {endif}
