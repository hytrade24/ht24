Merge Tool

Sie haben das Hersteller Datenbank Merge Tool aktiviert. Damit können Sie doppelte Hersteller zu einem zusammenfassen.

  1. Wählen Sie aus der unten stehenden Tabelle alle Hersteller aus, die zusammengefasst werden sollen und fügen Sie diese mit dem Button "Markierte Hersteller dem Merge Tool hinzufügen" dem Merge Tool hinzu.
  2. Wählen Sie anschließend den Ziel-Hersteller über die vorangestellte Auswhalmöglichkeit im Merge Tool
