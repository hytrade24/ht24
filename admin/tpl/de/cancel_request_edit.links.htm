Achtung!

Datenverlust möglich

Positionen, die Sie mit "Stornieren Leistung entziehen" bestätigen, können einen Datenverlust für Ihre Kunden zufolge haben!
Wenn Sie Z.B. ein Anzeigenpaket stornieren und die Leistung entziehen, hat das zufolge, dass alle Anzeigen, die diesem Paket zugeordnet wurden gelöscht werden!
