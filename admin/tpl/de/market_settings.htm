Einstellungen für den Marktplatz

Die folgenden Einstellungen sind für den Betrieb der Marktplatz Software ausgesprochen wichtig.
Achten Sie unbedingt auf die korrekte Schreibweise!
Fehlerhafte Einstellungen können dazu führen, dass der Betrieb des Marktplatzes stark eingeschränkt wird.

Einstellung Wert Beschreibung
Seitenname   Beschreibender Name Ihrer Webseite. z.B. "Mein neuer und genialer Anzeigenmarkt"
Dieser Name wird als sichtbarer Titel auf allen Seite verwendet
URL zu der Webseite   Adresse der Webseite inkl. dem Protokioll. z.B. "http://www.example.com"
Base Url (Unterverzeichnis)   Wird die Seite in einem Unterverzeichnis betrieben, geben Sie dieses hier an z.B. "/marktplatz/"
Absender Email   Diese Emailadresse wird verwendet um mit den Kunden in Kontakt zu treten. z.B. "nobody@example.com"
Standard-Währung   Global im Marktplatz verwendete Währung
Eingeschränkt sichtbare Anzeigen   Eingeschränkt sichtbare Anzeigen aktivieren/deaktivieren (z.B. B2B)
Miet-Anzeigen   Das Anbieten vermietbarer Anzeigen aktivieren/deaktivieren
Anzeigen manuell freischalten   Ein Administrator muss Anzeigen freigeben bevor diese öffentlich sichtbar werden.
Veranstaltungen manuell freischalten   Ein Administrator muss Veranstaltungen freigeben bevor diese öffentlich sichtbar werden.
Anbieter manuell freischalten   Ein Administrator muss Anbieter freigeben bevor diese öffentlich sichtbar werden.
Kontaktdaten des Absenders anzeigen   Bei Nachrichten und Anfragen über den Marktplatz werden die Kontaktdaten des Absenders dargestellt.
Kontaktdaten bei unbestätigten Verkäufen ausblenden   Bei Verkäufen werden die Kontaktdaten der Käufers/Verkäufers ausgeblendet, solange der Verkauf nicht bestätigt wurde.

Allgemeine Darstellung

Mit den folgenden Einstellungen verändern Sie das Verhalten der Marktplatz Software
maßgeblich. Sie können jede Einstellung jederzeit gefahrlos umstellen. Es wird
Ihnen jedoch dringend empfohlen nach jeder Änderung das Verhalten des Marktes zu
prüfen!

Einstellung Wert Beschreibung
Erweitertes HTML erlauben   HTML-Design z.B. beim Import aus ebay erlauben?
Artikelkauf   Aktiviert oder Deaktiviert die Möglichkeit Artikel zu kaufen/verkaufen
Kostenloses einstellen   Aktiviert oder Deaktiviert das kostenlose einstellen von Anzeigen.
Herstellerdatenbank aktivieren   Hersteller- und Produktdatenbank beim Einstellen von neuen Anzeigen aktivieren.
EAN Nummernsuche aktivieren   Eingabefeld für EAN-Nummer innerhalb der Suchmaske anzeigen
Kostenlose Bilder   Legt die Anzahl der kostenlosen Bilder für eine Anzeige fest.
Kostenlose Uploads   Verwendbare Uploads in einer Anzeige
Kostenlose Videos   Verwendbare Videos in einer Anzeige
Erlaubte Dateitypen   Erlaubte File Typen (Kommagetrennt)
Neue Anzeigen   Dauer in Tagen die ein Artikel als "Neu" angezeigt wird.
Neue Anzeigen auf der Startseite   Anzahl der neuen Anzeigen auf der Startseite
Neue TOP Anzeigen auf der Startseite   Anzahl der neuen Topanzeigen auf der Startseite
Erweiterete TOP Anzeigen   Erweiterte Einstellungen bei Top-Anzeigen aktivieren.
Cross selling Produkte   Anzahl der cross selling produkte unterhalb eines Artikels
Neue User auf der Startseite   Anzahl der neuen Benutzer (Willkommen) auf der Startseite
Anzahl der Listeneinträge im Shopverzeichnis   Anzahl der Einträge im Shopverzeichnis
Anonymer Artikelkauf   Erlaubt es nicht registrierten Besuchern Artikel auf dem Marktplatz zu kaufen
Umrechnung in andere Währung anzeigen   Stellt unter dem Artikelpreis die Umwandlung des Preis in andere Währungen dar.

Einstellungen für die Google Map

Hier können Sie einstellen, ob die Google Map bei den Listenansichten von
Anzeigen/Anbieter/Veranstaltungen angezeigt werden sollen.

Einstellung Wert Beschreibung
Google Maps API-Key (Server-Schlüssel) Ein Google API-Key für diesen Marktplatz. Gehen Sie auf https://console.developers.google.com/apis/credentials und erstellen Sie einen API-Schlüssel ("Server-Schlüssel").
Google Maps API-Key (Javascript-Schlüssel) Ein Google API-Key für diesen Marktplatz. Gehen Sie auf https://console.developers.google.com/apis/credentials und erstellen Sie einen API-Schlüssel ("Javascript-Schlüssel").
Google Map aktivieren   Wenn diese Option aktiv ist, dann wird die Google Map angezeigt. Wird diese Option nicht aktiviert, dann werden alle Einstellungen zu der Google Map ignoriert.
Bei Anbieter aktivieren   Map bei den Anbietern darstellen. Die Map wird bei der Listendarstellung und der Umkreissuche angezeigt.
Nur bei der Suche anzeigen   Wenn diese Option aktiviert wird, dann wird die Google Map nur in der Suche angezeigt. Wenn Sie wollen, dass die Map bei der Suche und bei der Listendarstellung angezeigt werden soll, dann deaktivieren Sie diese Option.
Bei Veranstaltungen aktivieren   Map bei den Veranstaltungen darstellen. Die Map wird bei der Listendarstellung und der Umkreissuche angezeigt.
Nur bei der Suche anzeigen   Wenn diese Option aktiviert wird, dann wird die Google Map nur in der Suche angezeigt. Wenn Sie wollen, dass die Map bei der Suche und bei der Listendarstellung angezeigt werden soll, dann deaktivieren Sie diese Option.
Maximale Zoom-Stufe für die Kartendarstellung Die Maximale Zoom-Stufe für die Kartendarstellung. Je größer der Wert, desto näher ist die Karte herangezoomt. (1-20)
