Texte bearbeiten

Hier können Sie Ihrem vorher ausgewählten Bild Text hinzufügen. Sie können mehrere Texte unabhänging voneinander platzieren, indem Sie mehrere Layer (Ebenen) erstellen. Für jeden Layer können Sie individuell Schriftart (aus den installierten), Schriftgröße, Schriftfarbe und Textposition angeben.

In das Bild klicken, um den aktuellen Text zu positionieren.

Bild Speichern: 
Für den Bildnamen sind nur folgende Zeichen erlaubt: a-z, 0-9, Bindestrich und Unterstrich. Alle Buchstaben werden automatisch in Kleinschreibung umgewandelt.
(Achtung! Falls Sie einen Namen angeben, der bereits existiert, werden für dieses Bild alle vorigen Einstellungen überschrieben.)
