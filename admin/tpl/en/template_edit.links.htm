Nutzen Sie den Datei- Editor, wie Sie es von Ihrem Textverarbeitungsprogramm her gewohnt sind. Sie können Texte formatieren, Bilder einfügen und Links setzen.
Sollte Ihnen die Bedeutung eines Buttons nicht ganz bewusst sein, fahren Sie kurz mit der Maus darüber. Es erscheint ein kleiner Hinweis- Text, welcher Ihnen die Funktion des Buttons erläutert.

Wenn Sie bereits Erfahrungen im Bereich HTML haben, können Sie auch direkt im "Quellcode" arbeiten. Drücken Sie hierzu auf "Quellcode". Sie können jederzeit wieder in den normalen Modus wechseln.

Sollte der Editor nicht geladen werden, oder aber einen Fehler verursachen, stellen Sie bitte sicher, dass JavaScript in Ihrem Browser aktiviert ist, und Ihr Popup- Blocker deaktiviert ist.
