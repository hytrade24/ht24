Nicht erlaubte HTML Tags

Hier haben Sie die Möglichkeit bestimmte HTML Tags von der Bunutzung auszuschließen. Diese können dann beim Schalten von Anzeigen nicht mehr verwendet werden!
Bitte geben Sie nur den namen des Tags an. z.B. "img" oder "object" Geben Sie bitte keine Zeichen wie ">,<" etc. ein.
Bitte beachten Sie dass, sehr viele Tags wieder geschlossen werden müssen. Wie z.B. <td>TEXT</td> Dieses können Sie separat vermerken.
Bei Tags wie <td>TEXT</td> können Sie entscheiden, was mit den Inhalten zwischen dem öffnenden und dem schließenden tag passieren soll.
