Editor Beispielseite

Diese Beispielseite dient der korrekten Einbindung des Editors in Ihre Anwendung. Bitte beachten Sie die benötigten Einstellungen

Art Ort Beschreibung
CHMOD 777 Upload Ordner Der Ordner muss von PHP schreibbar sein, da sonst keine Files hochgeladen werden können.
Upload Ordner anlegen Hauptverzeichnis Hier werden alles Files abgelegt, welche Ihre Benutzer über den Editor hochladen können.
Upload Pfad speichern Systemeinstellungen Hier müssen Sie den Upload Pfad speichern. Dieser wird dann vom Editor verwendet.
CHMOD 777 /fckeditor/fckstyles.xml Diese Datei wird von PHP geschrieben, und mit Ihren CSS-Definitionen gefüttert.
Recht editor_images erstellen Rollen & Rechte Mit diesem Recht bestimmen Sie welche User Bilder in welchem Umfang nutzen können.
