[[ translation : marketplace : my.imports.title :: Meine Imports ]]

[[ translation : marketplace : my.imports.help.intro ::

Über die Importschnittstelle haben Sie die Möglichkeit Artikel aus Ihren Quellsystemen, z.B. ERP-Systemen, Warenwirtschaften oder Onlineshops zu importieren. Der Martkplatz verfügt dafür über eine eigene Oberfläche. In der Übersicht, sieht er wie folgt aus:

  1. Import Vorlage anlegen
    Mit Importvorlagen stellen Sie eine Schnittstelle zwischen der Struktur Ihres Quellsystems und des Marktplatzes her. Sie ordnen Kategorien und Artikeleigenschaften zu
  2. Import Testlauf
    Vor dem eigentlichen Import können Sie einen Testlauf starten, der Ihnen mögliche Probleme und Fehler anzeigt
  3. Import der Datensätze
    Im Anschluss importieren Sie die Datensätze in den Marktplatz.
]]
