ebiz-trader Update

Anleitung für die Aktualisierung von Version {VERSION_FROM} auf {VERSION_TO}

In diesem Dokument werden die notwendigen Schritte für ein Update des ebiz-trader von Version {VERSION_FROM} auf {VERSION_TO} beschrieben. Grundlage für die Aktualisierung ist eine vollständige und funktionstüchtige Installation der Ausgangsversion.
Durch das Update gehen keine Daten oder Einstellungen verloren.

Wichtig!

  • Erstellen Sie ein Backup aller Dateien der aktuellen ebiz-trader Installation!
  • Erstellen Sie ein vollständiges Backup der gesamten Datenbank, in der Ihr ebiz-trader installiert wurde!!

Bei Rückfragen stehen wir Ihnen gerne zur Verfügung!

{PAGE_HEADER}

Vorbemerkung

Mit der Version 6.5.1 haben wir ein neues Updateprogramm eingeführt, um die Aktualisierung der Software weitestgehend automatisiert auszuführen und damit zu erleichtern.
Ihr Einschreiten ist nur an den Stellen erforderlich, an denen Sie manuell Änderungen ge- genüber der Standardversion vorgenommen haben.

Die folgende Updateanweisung zeigt und erklärt daher mögliche Eingriffe. Je nach Instal- lation werden bei Ihnen alle, nur ein Teil oder ggf. keine der beschriebenen Eingriffe not- wendig sein.

Wir empfehlen Ihnen daher eindringlich vorab eine Sicherungskopie anzulegen.

1) Vorbereitung

  • Öffnen Sie das Frontend und die Administration Ihrer Installation des ebiz-trader in einem Browser
  • Starten Sie ein FTP Programm und loggen sich auf Ihrem Webspace ein. Die folgende Anleitung geht davon aus, dass Sie sich im Hauptverzeichnis des ebiz-trader befinden.

2) Update-Anleitung

  1. Erstellen Sie ein Backup aller Dateien der aktuellen ebiz-trader Installation
  2. Erstellen Sie ein vollständiges Backup der gesamten Datenbank, in der Ihr ebiz-trader installiert wurde.
  3. Laden Sie alle Dateien aus dem Updatepaket ./Dateien/ per FTP in das Hauptverzeichnis des ebiz-trader hoch. Überschreiben Sie dabei schon vorhandene Dateien und Ordner.
  4. Achten Sie darauf, dass Sie in Ihrem FTP-Programm den Übertragungsmodus Binär ausgewählt haben.
  5. Rufen Sie die Administration Ihrer ebiz-trader Installation auf. Diese finden Sie i.d.R. unter http://www.ihredomain.de/admin/
  6. Sollten Sie noch nicht eingeloggt sein, melden Sie sich mit den Admin Zugangsdaten an.

{PAGE_HEADER}
  1. Folgen Sie den Anweisungen auf dem Bildschirm. Starten Sie das Update, indem Sie auf den Button „Update starten“ drücken.
  1. Wenn keine manuellen Eingriffe notwendig sein sollten, führt Sie das Updateprogramm durch alle weiteren Schritte. Hält das Programm an einem Schritt an, folgen Sie bitte den Anweisungen unter „Mögliche erforderliche Eingriffe Ihrerseits“

Mögliche erforderliche Eingriffe Ihrerseits

1) PHP Datei wurde angepasst / Schritt Dateien kopieren

In diesem Fall wurden Änderungen an PHP Dateien in Ihrer Installation erkannt, die nicht unserer Standardversion entsprechen.
Sie sehen nun eine Vergleichsoberfläche, in der Ihnen die erkannten Änderungen aufgezeigt werden.
Hier gibt es in der Regel 3 Ansichten:

Original-Version (vor dem Update)
In diesem Bereich sehen Sie die Datei ihrer aktuellen ebiz-trader Version ohne ihre Anpassungen.

Neue Version (nach dem Update)
In diesem Bereich sehen Sie die neue Version der Datei, wie Sie mit dem Update ausgeliefert wird. Vergleichen Sie diese mit der Original-Version um festzustellen welchen Änderungen im Zuge des Updates vorgenommen wurden.

Aktuelle Version (modifiziert)
In diesem Bereich sehen Sie aktuelle Version der Datei, wie Sie in ihrer ebiz-trader Version vorliegt. Vergleichen Sie diese mit der Neuen Version oder Original-Version um festzustellen welchen Änderungen Sie an dieser Datei vorgenommen haben.

{PAGE_FOOTER}
{PAGE_HEADER}

Die Bearbeitung erfolgt immer im linken (bei 2 Datei-Versionen) bzw. mittleren (bei 3 Datei- Versionen) Bereich. (Auch zu erkennen am helleren/weißen Hintergrund.) Sie können ihre Änderungen jederzeit speichern und zu einem späteren Zeitpunkt fortsetzen. Zwischen den Überschriften der Bereiche "Neue Version" und "Aktuelle Version" haben Sie die Möglichkeit diese Bereiche zu tauschen um, anstelle auf Basis der neuen Version, das Update auf Basis ihrer aktuellen Version vorzunehmen.

Wenn Sie die Änderungen vollständig zusammengeführt haben, klicken Sie auf den Button "Jetzt abschließen!" um die Änderungen engültig zu speichern. Sobald Sie dies machen wird die aktuelle Datei ihrer ebiz-trader Installation unwiderruflich durch die von ihnen zusammengeführte Datei ersetzt! Stellen Sie daher zuvor sicher, dass Sie auch wirklich alle nötigen Änderungen übertragen haben.

Sobald alle Dateien zusammengeführt wurden werden Sie automatisch gefragt ob das Update fortgesetzt werden soll. Mit einem Klick auf „ok“ (oder später auf „Update fortsetzen) geht das Update-Tool zum nächsten Schritt über.

Die Vergleichsansicht dient als nützliche Unterstützung, um gemachte Änderungen zu erkennen. Wir empfehlen Ihnen dennoch immer jede Änderung an unserer Standardversion zu dokumentieren!

3) E-Mail Template / Schritt Userdesign überprüfen

Es wurden Änderungen an E-Mail Templates gegenüber der Standardversion festgestellt, sodass das Updateprogramm diese nicht automatisch überschreiben kann. Ihnen wird daher der Inhalt des neuen Templates dargestellt. Bitte kopieren Sie diesen per Hand in das angegebene E-Mail Template und bestätigen Sie die Aktion mit „Update Änderungen ignorieren“



Abschluss des Updates

Das Update ist mit den o.g. Schritten abgeschlossen. Überprüfen Sie die neue Versionsnummer {VERSION_TO} auf der Startseite der Administration.
