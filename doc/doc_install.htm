Installation der ebiz-trader Software {VERSION_TO}

Folgende Schritte beschreiben, wie Sie ebiz-trader {VERSION_TO} auf einem Webserver installieren.

Vorbereitungen

MySQL-Datenbank
ebiz-trader benötigt eine mySQL Datenbank. Diese sollte bereits vor der Installation vorhanden sein. Sollten Sie diese noch nicht angelegt haben, so legen Sie diese bitte jetzt.

Domain
Der Domainname muss auf folgendes Root-Verzeichnis zeigen.

z.b. /HTML/ebiz-trader/

Diese Einstellung kann in der Regel über die Managementkonsolen Ihres Providers eingestellt werden.
Selbstverständlich können Sie unseren Marktplatz auch in einem Unterverzeichnis einer bestehenden Webseite installieren. Hierfür geben Sie dann bitte den kompletten Pfad wir folgt an.

z.b. /HTML/ihre/webseite/ebiz-trader/

Sollten Sie einen anderen Verzeichnis-Namen bevorzugen, so können Sie dieses gerne nach Ihren Vorstellungen anpassen. In unsere Dokumentation gehen wir jedoch von einer Installation im „ebiztrader“-Verzeichnis aus.

Installation:

Für User mit SSH-Zugang

  1. Laden Sie die gepackte Version in das Root-Verzeichnis Ihres Webserver.
    z.b. /HTML/
  2. Entpacken Sie die Datei mit folgendem Befehl.
    Durch das entpacken wird ein neues Unterverzeichnis mit dem Namen ebiz-trader angelegt.
    tar -xzf ebiz-trader_{VERSION_TO}tar.gz
  3. Ändern Sie die Gruppen und Besitzer für alle Dateien auf den Webbesitzer
    z.b. chown web54:web54 * -R

Per FTP-Upload

  1. Entpacken sie alle Dateien sowie Verzeichnisse auf Ihrer lokalen Festplatte
  2. Kopieren Sie die Dateien sowie Verzeichnisse auf den FTP-Server
Unsere Grafik zeigt alle Hauptverzeichnisse die Sie jetzt bereits per FTP-Client sehen sollte

Installationsprogramm ausführen

Rufen Sie über Ihren Browser ihre Domain auf, auf der sich die Installation des ebiz-traders befindet. Wenn ebiz-trader im Rootverzeichnis installiert wurde, dann führen Sie folgende URL aus
z.b. http://www.ihrewebseite.de

Sollten Sie ebiz-trader in einem Unterverzeichnis installieren wollen. Dann geben Sie bitte http://www.ihrewebseite.de/marktplatz/ in Ihrem Browser ein.

Sie sollten dann folgenden Bildschirm sehen:

Portaldetails eintragen

Geben Sie in der folgenden Maske die Details zu Ihrer Webseite ein.

Seitenname:    Geben Sie hier den Projekt Name an. z.B. „benola Marktplatz“
Projekt-Url:    Unter welcher URl soll Ihr Projekt erreichbar sein. Z.b. http://www.benola.de
Base-Url:    Geben sie das Unterverzeichnis an in dem ebiz-trader später laufen soll
                     Wird der Marktplatz auf der Hauptdomain betrieben, so lass Sie bitte „/“ stehen
Admin-Password :    Geben Sie hier das Password für den Administrator ein
Admin-Emailadresse:    Geben Sie hier die E-Mail-Adresse des Administrators ein

Einrichtung der mySQL-Datenbank

Die Zugangsdaten zu Ihrem mySQL-Datenbank-Server sollten Sie von Ihrem Provider erhalten haben.

Cronjob installieren

ebiz-trader führt viele Aktionen im Hintergrund aus. Aus diesem Grund benötigen wir einen Cronjob der im Idealfall minütlich ausgeführt werden sollte.

Tragen Sei folgenden Befehl in die Cron-Tab ein.

/usr/bin/php5 /var/www/web60/html/ebiz-trader/cron/cronjob.php > /dev/null 2>&1

Achten Sie auf die richtigen Pfadangaben!!
